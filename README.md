# Проект: Sound from Pixels (Data Sonification App)
## Автор - Гимранов Равиль Римович
## Ссылка на репозиторий https://github.com/RavilG882/sound-from-pixels

## Идея проекта
Создание веб-приложения для преобразования визуальных данных в аудио. 
Пользователь рисует произвольную непрерывную кривую на интерактивном холсте. Программа считывает матрицу пикселей, извлекает координаты нарисованной линии и с помощью алгоритмов цифровой обработки сигналов (DSP) синтезирует из них полноценный звуковой `.wav` файл. Горизонтальная ось X интерпретируется как время, а вертикальная ось Y конвертируется в частоту звуковой волны (Гц).

## План реализации

### Итерация 1
1. **Модуль извлечения координат:** Настройка компонента `streamlit-drawable-canvas` и написание парсера, который переводит полученную RGBA-матрицу изображения в два одномерных массива (координаты X и Y), фильтруя пустые пиксели и обрабатывая разрывы линий.
2. **Базовый DSP-синтезатор:** Разработка класса-генератора, который производит сплайновую интерполяцию полученных точек и с помощью векторных вычислений генерирует PCM-массив (чистую синусоиду) на основе маппинга оси Y в слышимый диапазон частот.

### Итерация 2
1. **Дополнения:** Контроль времени и скорости звучания.
2. **Шаблонные мелодии:** Генерация мелодии из готового шаблона

## Библиотеки - scipy, streamlit
